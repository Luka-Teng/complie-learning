### 如何运行
`ts-node run.ts`

### 目前支持语法

### 注意事项，语法不需要以;结尾

### 1. Identifer
```
表示变量名，如id
```

### 2. number
```
表示数字
```

### 3. 赋值
```
a = c = 222
```

### 4. 声明

#### let声明
```
let a = 1
```

### 5. 二元表达式

#### 四则运算
```
a + 2 - 3 * 4 / 4
```

#### 逻辑元算(或、并)
```
a || b && c
```

#### 暂时不支持字符串、布尔等数据类型，if、function等文法
